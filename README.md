# Customer Segmentation Analysis using K-Means Clustering

## Project Overview

This project performs Customer Segmentation Analysis using the K-Means Clustering algorithm. The objective is to group customers based on their purchasing behaviour using RFM (Recency, Frequency, Monetary) analysis. These customer segments help businesses create targeted marketing strategies.

## Objectives

- Load and inspect the dataset
- Clean missing and duplicate values
- Perform descriptive statistics
- Create RFM features
- Standardize the data
- Apply K-Means Clustering
- Determine optimal K using the Elbow Method
- Visualize customer clusters
- Profile each customer segment
- Recommend marketing strategies

## Tools & Technologies

- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-learn
- Jupyter Notebook

## Project Workflow

1. Data Loading
2. Data Cleaning
3. Exploratory Data Analysis
4. Feature Engineering (RFM)
5. Data Standardization
6. Elbow Method
7. K-Means Clustering
8. Cluster Visualization
9. Cluster Profiling
10. Marketing Recommendations

## Visualizations

- Elbow Method Graph
- Scatter Plot (Frequency vs Monetary)
- Scatter Plot (Recency vs Monetary)
- Customers per Cluster Bar Chart

## Customer Segments

### Cluster 0
- High-value customers
- Frequent buyers
- High spending

**Recommendation:** VIP rewards and exclusive offers.

### Cluster 1
- Low-value customers
- Infrequent buyers

**Recommendation:** Discount coupons and promotional emails.

### Cluster 2
- Medium-value customers

**Recommendation:** Cross-selling and product recommendations.

### Cluster 3
- Inactive customers

**Recommendation:** Re-engagement campaigns and seasonal offers.

## Conclusion

Customer segmentation using K-Means helps businesses understand customer behaviour and improve marketing strategies through personalized campaigns.
